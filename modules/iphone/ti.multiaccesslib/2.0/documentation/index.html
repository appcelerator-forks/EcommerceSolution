media library multi access Module

Description

This module is used to present media picker as native iOS photo library. It shows media picker as a modal media picker and enables to select multiple images. It supports browsing through the available albums. It also provides flexibility to choose maximum number of images to be selected and also to choose to return full resolution Image or full screen image.

Accessing the media library multi access Module

To access this module from JavaScript, you would do the following:

var media library multi access = require("ti.multiAccessLib");

The media library multi access variable is a reference to the Module object.
